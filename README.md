# Wireshark Seq-Ack TCP Analysis

## Description

This project demonstrates the analysis of TCP sequence and acknowledgement numbers using Wireshark. It includes the study of the TCP three-way handshake and the simulation of various TCP-based attacks using hping3. The experiment helps in understanding packet-level communication and identifying abnormal network behavior.

## Objectives

* To analyze TCP sequence and acknowledgement numbers
* To study the TCP three-way handshake
* To demonstrate TCP-based attacks such as SYN flood, RST attack, data flood, FIN-ACK flood, and reflection attack
* To understand network communication and security concepts

## Technologies Used

* Wireshark
* hping3

## Files Included

* `pcap_files/`
  Contains packet capture files for:

  * SYN flood
  * RST attack
  * Data flood
  * FIN-ACK flood
  * Reflection attack

* `documentation.pdf`
  Contains detailed procedure, code, and analysis

## How to Perform

1. Capture network traffic using Wireshark
2. Apply TCP filters to analyze packets
3. Use hping3 commands to generate attack traffic
4. Observe packet behavior in Wireshark

## Conclusion

The project demonstrates TCP communication using sequence and acknowledgement numbers and explains how different TCP attacks can be identified through packet analysis. It highlights the importance of packet inspection in network security.
